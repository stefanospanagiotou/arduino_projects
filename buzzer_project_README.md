Αυτό το project κάνει το Arduino να λειτουργεί σαν σειρήνα αστυνομίας.
Όταν πατήσεις το κουμπί, οι δύο λυχνίες LED αναβοσβήνουν εναλλάξ και ο βομβητής (buzzer) βγάζει ήχο σειρήνας.

Όταν ξαναπατήσεις το κουμπί, η σειρήνα σταματά, οι λυχνίες σβήνουν και δεν ακούγεται ήχος.

Οι λυχνίες είναι σβηστές όταν το σύστημα είναι κλειστό και ανάβουν μόνο όταν η σειρήνα λειτουργεί.
