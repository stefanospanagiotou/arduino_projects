Αυτό το πρόγραμμα για Arduino κάνει δύο λυχνίες LED να αναβοσβήνουν εναλλάξ —
όταν η μία είναι αναμμένη, η άλλη είναι σβηστή.
Μετά από μισό δευτερόλεπτο, αλλάζουν ρόλους, δημιουργώντας ένα φωτεινό εφέ “ping-pong”.

Η διαδικασία επαναλαμβάνεται συνεχώς, έτσι οι δύο λυχνίες φαίνονται να αναβοσβήνουν η μία μετά την άλλη.

Μπορείς να αλλάξεις την ταχύτητα αναβοσβήματος τροποποιώντας τον αριθμό στη γραμμή delay(500)
